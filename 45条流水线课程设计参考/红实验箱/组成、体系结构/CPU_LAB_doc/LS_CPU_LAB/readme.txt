本目录给出了LS-CPU_EXB-001实验箱的配套《cpu设计与体系结构实验指导手册》对应的参考设计的ISE工程。

所有实验参考设计的工程都是在windows7下的ISE13.2版本下建立的，也可以移植到高版本上运行

   |
   |---4_alu/             《实验四  ：ALU模块实现         》的参考设计的ISE project
   |
   |---5_memory/          《实验五  ：存储器实现          》的参考设计的ISE project
   |
   |---6_single_cycle_cpu/《实验六  ：单周期CPU实现       》的参考设计的ISE project
   |
   |---7_multi_cycle_cpu/ 《实验七  ：多周期CPU实现       》的参考设计的ISE project
   |
   |---8_pipeline_cpu/    《课程设计：静态5级流水线CPU实现》的参考设计的ISE project
